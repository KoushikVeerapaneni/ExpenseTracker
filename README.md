# ExpenseTracker
A modern expense tracker app built using Python and Tkinter.

Features
- Add, edit, and delete expenses easily  
- Automatically calculate total expenses  
- Lightweight and easy to run locally  
- Clean, minimal, and responsive layout  
- Export data to CSV  
- Category-wise summary view  

Tech Stack
- Python 3.x
- Tkinter (for UI)
- SQLite3 (for data storage)
- tkcalendar (for date picker)
